# Proof-of-Courier-Fee
快递运费凭证模板
1:1 还原菜鸟官方运费支付凭证，全参数自定义 + 一键导出保存，解决批量面单无法单独开具运费证明的刚需工具

📌 项目介绍
日常使用菜鸟批量寄件、月结账号、企业面单时，平台无法针对单个快递单号单独生成运费支付凭证，报销、申诉、举证、对账时极度不便。
本项目纯前端复刻 菜鸟官方运费支付凭证页面 UI，做到视觉 1:1 还原，支持所有字段自由自定义填写，无需后端、无需登录、无数据上传，本地一键生成并保存标准运费证明图片，完美适配个人报销、售后举证、平台申诉、对账归档等场景。

✨ 核心功能
• 🎨 1:1 官方还原
完全复刻菜鸟原版运费支付凭证样式、排版、字体、配色，和官方导出凭证无视觉差异。
• 📝 全字段自定义
快递单号、寄收地址、寄件时间、支付金额、运费明细、订单状态、支付渠道等 全部内容可自由修改。
• 💾 一键本地保存
前端一键导出高清图片，无水印、无压缩，本地生成，零服务器上传，隐私安全。
• ⚡ 轻量免部署
纯 HTML+CSS+JS 静态页面，下载即用、打开即用，支持本地离线使用，可部署 Github Pages 长期在线。
• 🧩 适配所有特殊场景
完美解决：月结账号、批量面单、企业寄件、合并订单 无法开具单笔运费证明 的行业痛点。

🎯 适用场景
• 个人/学生 运费报销、校园报销、平台售后申诉
• 电商商家 订单举证、退换货运费凭证留存
• 企业对账、财务归档、售后维权凭证制作
• 所有「官方无法单独导出单笔快递运费证明」的场景

🚀 使用方法
1. 克隆 / 下载本仓库静态文件
2. 直接打开 index.html 即可使用
3. 根据需求自定义修改所有表单内容
4. 点击保存按钮，一键导出高清运费支付凭证

🔒 项目优势
1. 本地离线运行：所有数据仅保存在本地，不上传任何信息，安全无风险
2. 零成本使用：无需注册、无需会员、无广告、无付费功
3. 高度仿真：细节对齐官方原版，凭证正规可用
4. 极致便捷：打开即用，无需部署环境，小白也能轻松操作
